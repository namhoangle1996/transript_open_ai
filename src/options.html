Extension Settings

Prompt for Summary

You can customize the prompt for summary.

Input format:

Prompt: {{Prompt}}

Title: {{Title}}

Transcript: {{Transcript}}

Example Prompts:
  • What's key takeaways from the following?
  • Extract the gist of the following.
  • Summarize the following for a 6th grade audience.
Follow-up Prompts

You can set up (up to 10) follow-up prompts. These follow-up prompts will be shown near the prompt inbox.
